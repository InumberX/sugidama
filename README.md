## Development

### Installing packages

```shell
npm install
```

### Execution of development tasks

Execute the following command.

```shell
npm run dev
```

The following URL will take you to the screen.

http://localhost:5173/

- Press `Ctrl + C` to stop

### Syntax Check

#### Check

```shell
npm run lint
```

#### Check and Format

```shell
npm run lint-fix
```

### Formatter

#### Check

```shell
npm run format
```

#### Check and Format

```shell
npm run format-fix
```

### Storybook
